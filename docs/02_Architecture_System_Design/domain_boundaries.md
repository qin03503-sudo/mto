# Domain Boundaries

## Auth Domain

Responsibilities:

- Users
- Roles
- Permissions
- Session or token management

## Master Data Domain

Responsibilities:

- Scopes
- Parts
- Materials
- Units
- MTO versions
- MTO rows

## Offer Domain

Responsibilities:

- Offers/projects
- Project scopes
- Lines
- Line parts
- Offer status

## Material Pricing Domain

Responsibilities:

- Default material prices
- Project-specific material prices
- Price overrides
- Price history

## Calculation Domain

Responsibilities:

- MTO row calculation
- Part unit price calculation
- Line, scope, and offer totals
- Calculation snapshots
- Outdated status

## Import Domain

Responsibilities:

- Excel staging
- Data validation
- Import versioning
- Error reports
