# Pages And Routing

Suggested routes:

- `/login`
- `/offers`
- `/offers/new`
- `/offers/:id/overview`
- `/offers/:id/material-prices`
- `/offers/:id/scopes-lines`
- `/offers/:id/calculation`

Route responsibilities:

| Route | Responsibility |
| --- | --- |
| `/offers` | List, search, and open offers. |
| `/offers/new` | Create a new offer. |
| `/offers/:id/overview` | Show offer metadata and status. |
| `/offers/:id/material-prices` | View and edit project material prices. |
| `/offers/:id/scopes-lines` | Select scopes, create lines, and add parts. |
| `/offers/:id/calculation` | Run calculation and view results. |
