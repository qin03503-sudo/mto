# Roadmap Phases

## Phase 1: Database And Master Data

- Import initial data from the current Excel workbook.
- Normalize material names.
- Normalize units.
- Resolve duplicate material prices.
- Create and validate Material, Scope, Part, and MTO Master tables.
- Validate Excel totals after import.

## Phase 2: Offer And Project Material Prices

- Offer list and new offer form.
- Copy default material prices into project-specific material prices when an offer is created.
- Allow project material prices to be edited without changing base material prices or other offers.

## Phase 3: Scope, Line, And Part Entry

- Scope selection.
- Line creation.
- Part and quantity entry.

## Phase 4: Calculation Engine

- Project material price copy.
- Basic calculation engine.
- Output grouped by scope, line, and part.
- Line, scope, and project totals.

## Phase 5: Output Report

- Pivot-like output table.
- Summary totals.
- Preparation for future Excel or PDF export.

## Future: Pricing Control

- Material price override history.
- Calculation outdated status.
- Calculation snapshots and drill-down.
- MTO version management.

## Future: Reporting And Export

- PDF export.
- Excel export/import.
- Customer-facing report template.
- Profit, overhead, tax, discount, transport, and installation fields.
