# Glossary

| Term | Meaning |
| --- | --- |
| MTO | Material Take-Off. Master list of materials and consumption values used to calculate busduct part prices. |
| Offer | A pricing proposal or project quotation created for a customer. |
| Scope | Electrical/current scope, such as `1000A AL` or `1600A AL`. |
| Line | A route/path inside a project scope. Each line contains multiple parts. |
| Part | A busduct component used in a line, such as `FEEDER-1000`, `Flexible connection`, or `TOP OFF BOX 250`. |
| Material | Raw material, operation, or small component used inside a part. |
| Value | Main consumption coefficient used in most MTO calculations. |
| Qty | Quantity. Used for parts inside a line and for some MTO row calculation exceptions. |
| Unit Price | Price per material unit or calculated price per part, depending on context. |
| Total Price | Calculated total price for an MTO row, line part, line, scope, or offer. |
| AL | Aluminum conductor type. |
| CU | Copper conductor type, expected future scope type. |
| Snapshot | Stored calculation result used to preserve historical offer pricing. |
